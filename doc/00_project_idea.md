# Идея проекта

## Проблема
Пользователям сложно искать фильмы по описанию сюжета среди тысяч вариантов.

## Цель проекта
Построить RAG-систему, которая по запросу пользователя находит релевантные
фильмы из базы TMDB и показывает их описания с указанием источника.

## Для кого
Для любителей кино, которые ищут фильм по теме или сюжету.

## Какие данные
Источник: Kaggle — Top 10K TMDB Movies (ahsanaseer/top-rated-tmdb-movies-10k)
Масштаб: 9979 фильмов с описаниями
Формат: CSV с полями id, title, genre, overview, release_date, vote_average

## Критерий успеха
- 3 вопроса дают правильный ответ с указанием фильма-источника
- 1 вопрос получает корректный отказ
- Все тесты зелёные
