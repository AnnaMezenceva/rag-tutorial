# Данные и назначение репозитория

## Назначение репозитория

**Для кого:** студенты, изучающие RAG-системы.

**Что демонстрирует:** полный offline-pipeline: CSV → документы → чанки → TF-IDF индекс → поиск → demo-ответ с источниками в Streamlit UI.

**Границы MVP:** поиск по словам (TF-IDF), без embeddings и LLM, demo-режим без внешних API.

## Источники данных

| Источник | Файл в проекте | Комментарий |
|----------|----------------|-------------|
| [Top 10K TMDB Movies](https://www.kaggle.com/datasets/ahsanaseer/top-rated-tmdb-movies-10k) | `data/raw/top10K-TMDB-movies.csv` | 10 000 фильмов с описаниями сюжетов |
| Подготовленный корпус | `data/raw/datasets.json` | 9979 записей, генерируется скриптом |

## Что индексируем

| Поле | Индексируется? | Где используется |
|------|:--------------:|------------------|
| text (title + genre + overview) | **Да** | TF-IDF матрица, поиск, demo-ответ |
| name | Нет (метаданные) | UI, подпись документа |
| doc_id | Нет (метаданные) | UI, идентификатор записи |
