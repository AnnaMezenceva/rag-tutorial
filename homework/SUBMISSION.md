## Данные
Источник: [Top 10K TMDB Movies](https://www.kaggle.com/datasets/ahsanaseer/top-rated-tmdb-movies-10k)
Документов: 9979 фильмов, чанков: 16187.
К каждому фильму автоматически добавлены русскоязычные ключевые слова (жанры и тематические теги) для поддержки поиска на русском языке.

## Что реализовано
- Полный RAG-pipeline: CSV → datasets.json → documents.jsonl → chunks.jsonl → TF-IDF индекс → поиск → demo-ответ
- Автоматическое извлечение русских ключевых слов из английских описаний (40+ тематических тегов с учётом морфологии)
- Streamlit UI с настройками (порог score, top-k, история запросов)
- 5 demo-вопросов (4 релевантных + 1 negative)
- 11 тестов (pytest) — все зелёные
- Документация: 5 файлов планирования + DATA.md

## Demo-вопросы
1. «Какой фильм про побег из тюрьмы?» → The Shawshank Redemption и др.
2. «Какие фильмы о космосе?» → научная фантастика
